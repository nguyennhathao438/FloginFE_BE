# 🚀 FloginFE_BE

**FloginFE_BE** là một ứng dụng web **Fullstack** hoàn chỉnh được phát triển theo phương pháp **Test-Driven Development (TDD)**.  
Dự án bao gồm **Frontend (React)** và **Backend (Spring Boot)**, với đầy đủ chức năng **đăng nhập** và **quản lý sản phẩm (CRUD)**.

---

## 🧩 Tính năng chính

- 🔐 **Login:** Hệ thống đăng nhập với kiểm tra đầu vào (validation) đầy đủ  
- 📦 **Product Management:** Quản lý sản phẩm (thêm, sửa, xóa, xem chi tiết)  
- 🧪 **Testing:** Áp dụng phương pháp **TDD (Test-Driven Development)** cho cả frontend và backend

---

## ⚙️ Backend

| Thành phần | Phiên bản / Công nghệ |
|-------------|------------------------|
| **Framework** | Spring Boot `3.5.7` |
| **Ngôn ngữ** | Java `25` |
| **Testing** | JUnit 5, Mockito |
| **Build Tool** | Maven |
| **ORM** | Spring Data JPA |
| **CSDL** | MySQL |
| **Mục tiêu** | Xử lý logic nghiệp vụ, API RESTful, xác thực người dùng |

---

## 💻 Frontend

| Thành phần | Phiên bản / Công nghệ |
|-------------|------------------------|
| **Framework** | React `19.2.0` |
| **Testing** | React Testing Library `16.3.0`, Jest |
| **HTTP Client** | Axios |
| **Styling** | CSS3 với hiệu ứng animations |
| **Mục tiêu** | Giao diện người dùng, gọi API backend, hiển thị dữ liệu sản phẩm |
